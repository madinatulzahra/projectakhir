🪵 Rizz Furniture

Rizz Furniture adalah website company profile dan katalog produk digital berbasis PHP native, MySQL, dan Bootstrap 5 untuk usaha furniture dan interior modern.

Website ini memiliki halaman publik seperti Home, Tentang, Produk, dan Kontak, lengkap dengan katalog produk, WhatsApp order otomatis, testimoni pelanggan, timeline proses kerja, Google Maps, serta desain modern dan responsif.

Dilengkapi juga dengan panel admin untuk mengelola produk menggunakan DataTables, upload gambar dengan preview, autentikasi login, dan sistem keamanan dasar seperti prepared statement, session checking, serta proteksi XSS dan SQL Injection.

Teknologi: PHP · MySQL · Bootstrap 5 · SweetAlert2 · DataTables · AOS.js · Font Awesome
